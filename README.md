# CRM Мастер

**Живое демо:** https://pomah4uk.github.io/Beauty-Crm/

Универсальная CRM для учёта клиентов, записей и финансов. Работает в браузере, данные на телефоне, офлайн, без регистрации.

## Возможности

- Управление клиентами (имя, телефон, день рождения, рейтинг)
- Записи на услуги (услуги можно редактировать)
- Расходы
- Статистика (выручка, затраты, прибыль за месяц)
- Напоминание о давних клиентах
- Тёмная тема
- Бэкап (экспорт/импорт)
- PWA — установка на телефон

## Технологии

HTML, CSS, JS, PWA, localStorage
